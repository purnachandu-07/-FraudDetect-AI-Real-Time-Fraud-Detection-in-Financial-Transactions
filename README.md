# FraudDetect-AI-Real Time Fraud Detection in Financial Transactions
AI-powered fraud detection system for identifying suspicious financial transactions in real time.
| Feature                     | Details                                                                                  |
| --------------------------- | ---------------------------------------------------------------------------------------- |
| Real-Time Fraud Detection   | Detects fraudulent transactions instantly using Machine Learning models                  |
| Machine Learning Prediction | Uses XGBoost and Scikit-learn algorithms to classify transactions as fraud or legitimate |
| Live Transaction Monitoring | Tracks and updates transaction activity in real time                                     |
| WebSocket Alerts            | Sends instant fraud alerts without refreshing the dashboard                              |
| Secure Authentication       | JWT-based login system with password encryption using bcrypt                             |
| Transaction Management      | Stores, manages, and monitors all transaction records                                    |
| Interactive Dashboard       | Displays transaction status, fraud alerts, analytics, and monitoring data                |
| Database Integration        | Uses SQLite with SQLAlchemy ORM for secure and structured data storage                   |
| REST API Architecture       | FastAPI-based APIs for smooth frontend-backend communication                             |
| Seed Data Generation        | Generates sample users and transaction data for testing                                  |
| Modular Backend Structure   | Organized backend using routers, ML modules, and service separation                      |
| Responsive Frontend         | React + Tailwind CSS based UI accessible on desktop and mobile devices                   |
| Real-Time Communication     | WebSocket integration for live updates and notifications                                 |
| Fast Processing             | Optimized backend using FastAPI asynchronous capabilities                                |
| Local System Deployment     | Can run completely on local machine without cloud dependency
